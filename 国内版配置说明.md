# 分享墙 · 国内版配置（无需换线路）

国内网络访问 **Supabase** 经常超时或被阻断。  
**国内版**使用 [LeanCloud](https://console.leancloud.cn) 中国节点，**不需要 VPN**，手机电脑可远程同步。

---

## 第一步：注册 LeanCloud

1. 打开 https://console.leancloud.cn 注册
2. 创建应用（选 **开发版** 免费即可）
3. 存储区域选 **华北 / 华东**（国内节点）

## 第二步：创建数据表

1. 左侧 **数据存储** → **结构化数据** → **创建 Class**
2. 名称填：`SharePost`（区分大小写）
3. 创建后点进 `SharePost` → **权限** → **Class 权限**
4. 勾选：**find、get、create、update、delete** 均对 **所有用户** 开放  
   （小圈子使用；勿公开到搜索引擎）

无需手动建字段，发布时会自动创建 `boardId`、`text`、`link`、`imageUrl` 等。

## 第三步：开启文件存储（图片）

1. 左侧 **数据存储** → **文件**
2. 确认文件服务已开启（默认有）
3. **设置** → 允许 **所有用户** 上传/读取（与 Class 权限类似）

## 第四步：复制密钥

**设置** → **应用凭证** → **应用 Keys**：

| 复制项 | 填到 leancloud-config.js |
|--------|--------------------------|
| AppID | `appId` |
| AppKey（客户端，非 Master） | `appKey` |
| REST API 服务器地址 | `serverURL` |

`serverURL` 形如：`https://xxxxx.api.lncld.cn`（必须是 `.cn` 国内地址）

## 第五步：写入配置并发布

编辑 GitHub 仓库 **`leancloud-config.js`**：

```javascript
window.SHARE_BOARD_LC = {
  appId: "你的AppID",
  appKey: "你的AppKey",
  serverURL: "https://xxxxx.api.lncld.cn",
  board: "main",
};
```

Commit 后等 GitHub Pages 更新 1～2 分钟。

## 第六步：访问国内版

```
https://你的用户名.github.io/share-board-Nanbian525/分享墙-国内版.html
```

或打开首页（已默认跳转国内版）。

手机、电脑打开**同一链接**，房间号一致即可同步。

---

## 与 Supabase 版的区别

| | 国内版（LeanCloud） | 云端版（Supabase） |
|--|---------------------|-------------------|
| 国内访问 | ✅ 稳定 | ❌ 常超时 |
| 需 VPN | 否 | 有时需要 |
| 配置平台 | console.leancloud.cn | supabase.com |

你已配置的 Supabase 可保留；国内日常使用建议 **国内版**。
