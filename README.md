# 分享墙

简洁的协作发布板：发表文字、链接与图片，随时编辑或删除。

## 浏览器直接打开（推荐，无需安装）

双击 **`分享墙.html`** 或 **`打开分享墙.bat`** 即可使用。

- 数据保存在本浏览器，换电脑需用「导出/导入备份」同步
- 适合个人使用或偶尔与他人交换内容

## 服务器版（多人实时共用一面墙）

## 快速启动（Python）

1. 确保已安装 [Python 3](https://www.python.org/)（3.10 及以上）
2. 在本目录打开终端，执行：

```bash
pip install -r requirements.txt
python app.py
```

3. 浏览器打开：**http://localhost:3000**

### 可选：Node.js 方式

若已安装 Node.js，也可执行 `npm install` 后 `npm start`（使用 `server.js`）。

## 与他人共用

- **同一 WiFi / 局域网**：启动后终端会提示地址；他人用手机或电脑访问 `http://你的电脑IP:3000` 即可（需放行 Windows 防火墙对该端口的访问）。
- **公网访问**：可将项目部署到云服务器（如阿里云、腾讯云），或使用内网穿透工具（如 ngrok、frp）将本地 3000 端口映射到公网。

## 功能

- 发表文字、链接、图片（至少填一项）
- 卡片式展示全部内容
- 随时编辑、删除
- 数据保存在本地 `data/board.db`，图片在 `uploads/` 目录

## 目录说明

| 路径 | 说明 |
|------|------|
| `server.js` | 后端服务 |
| `public/` | 前端页面 |
| `data/` | 数据库（自动生成） |
| `uploads/` | 上传图片（自动生成） |
